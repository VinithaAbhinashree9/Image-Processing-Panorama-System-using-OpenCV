# Image-Processing-Panorama-System-using-OpenCV

---

 Image Processing & Panorama Creation System using Python, OpenCV, and NumPy with thresholding, noise removal, edge detection, feature detection, and panorama stitching.


A Computer Vision project developed using Python, OpenCV, and NumPy that performs advanced image processing operations and panorama stitching.

This project demonstrates important image processing concepts used in real-world photo editing applications.

---

#  Features

##  Image Enhancement
- Convert image to grayscale
- Apply thresholding
- Adjust brightness

##  Noise Removal
- Remove noise using Gaussian Blur

## Edge Detection
- Detect object boundaries using Canny Edge Detection

##  Feature Detection
- Detect important feature points using ORB

##  Panorama Creation
- Stitch multiple images into a panorama
